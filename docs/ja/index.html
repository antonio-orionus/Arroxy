できること

期待されることはすべて、面倒は一切なく。

URL を貼り付けて、画質を選んで、ダウンロードをクリック。それだけ。

最大 4K UHD

2160p、1440p、1080p、720p — YouTube が提供するあらゆる解像度に加え、MP3、AAC、Opus の音声のみも対応。

60 fps と HDR を維持

高フレームレートと HDR ストリームは YouTube がエンコードしたまま — 画質劣化なし。

複数同時に

好きなだけ動画をキューに追加できます。ダウンロードパネルが各進捗を並行して表示。

自動アップデート

Arroxy は yt-dlp と ffmpeg を裏で常に最新に保ちます — YouTube の変更にも対応。

9 言語対応

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी — あなたの言語を自動検出。

マルチプラットフォーム

Windows、macOS、Linux のネイティブビルド — インストーラー、ポータブル、DMG、AppImage。

字幕は思いのままに

手動または自動生成の字幕を SRT、VTT、ASS で取得 — 動画と並べて保存、ポータブルな .mkv に埋め込み、または Subtitles/ フォルダに整理。

実際の動作

明快さのために設計、雑然さは排除。

プライバシー

Arroxy がしないこと。

ほとんどの YouTube ダウンローダーは、いずれ Cookie を要求します。Arroxy は決して要求しません。

ログイン不要

Google アカウント不要。期限切れのセッションもなし。アカウントが警告される心配ゼロ。

Cookie 不要

Arroxy はブラウザと同じトークンを要求するだけ。何もエクスポートせず、何も保存しません。

テレメトリーなし

アナリティクス一切なし。ダウンロード、履歴、ファイルはすべてあなたのデバイス内に留まります。

サードパーティサーバーなし

全パイプラインが yt-dlp + ffmpeg でローカル実行。ファイルがリモートサーバーに触れることはありません。

インストール

チャンネルを選んでください。

直接ダウンロード、または主要パッケージマネージャー経由 — すべてリリースごとに自動更新。

Winget Windows

Windows 10/11 推奨。システムと一緒に自動更新。

winget install AntonioOrionus.Arroxy

Scoop Windows

Scoop バケット経由のポータブルインストール。管理者権限不要。

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Cask を tap してワンコマンドでインストール。ユニバーサルバイナリ (Intel + Apple Silicon)。

brew tap antonio-orionus/arroxy && brew install --cask arroxy
