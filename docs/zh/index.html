它能做什么

一切如你所愿，毫无阻碍。

粘贴 URL，选择画质，点击下载。就这么简单。

高达 4K UHD

2160p、1440p、1080p、720p — YouTube 提供的每一种分辨率，外加 MP3、AAC、Opus 纯音频。

保留 60 fps 与 HDR

高帧率与 HDR 流原样直通，正如 YouTube 编码的那样 — 零画质损失。

多任务并行

想排多少视频就排多少。下载面板并行显示每个任务的进度。

自动更新

Arroxy 在后台保持 yt-dlp 与 ffmpeg 最新 — 应对 YouTube 的每一次变化。

9 种语言

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी — 自动检测你的语言。

跨平台

Windows、macOS、Linux 原生构建 — 安装包、便携版、DMG 或 AppImage。

字幕，随你所愿

以 SRT、VTT 或 ASS 获取手动或自动生成的字幕 — 保存到视频旁边、嵌入便携的 .mkv，或整理到 Subtitles/ 子文件夹。

内置 SponsorBlock

跳过或标记赞助商、片头、片尾、自我推广等片段 — 用 FFmpeg 剪除或直接添加章节。按类别自由选择。

剪贴板自动填写

在任何地方复制 YouTube 链接，切换回应用时 Arroxy 立即检测 — 确认提示让你保持控制。在高级设置中开启或关闭。

自动清理 URL

粘贴的 YouTube 链接会自动剥除跟踪参数（si、pp、feature、utm_*、fbclid、gclid 等），并解包 youtube.com/redirect 跳转链接 — URL 字段始终显示规范链接。

最小化到托盘

关闭窗口后 Arroxy 将缩入系统托盘，下载在后台持续运行 — 单击托盘图标可恢复窗口，或通过托盘菜单退出应用。

隐私

Arroxy 不会做什么。

大多数 YouTube 下载器迟早会向你索取 Cookie。Arroxy 永远不会。

无需登录

无需 Google 账号。没有会话过期问题。账号被标记的风险为零。

无 Cookie

Arroxy 请求的只是浏览器同样会请求的令牌。不导出，不保存。

无遥测

零分析。你的下载、历史和文件全部留在本地设备 — 就这样。

无第三方服务器

整个流程通过 yt-dlp + ffmpeg 在本地运行。文件永远不会经过远程服务器。

FAQ

常见问题

能下载哪些画质？
YouTube 提供的都行 — 4K UHD（2160p）、1440p QHD、1080p Full HD、720p、480p、360p 以及纯音频。高帧率（60 fps、120 fps）和 HDR 内容原样保留。Arroxy 列出所有可用格式，让你精准挑选。
真的免费吗？
真的。MIT 许可证。没有付费版、没有功能门槛。
Arroxy 支持哪些语言？
开箱即用支持九种：English、Español（西班牙语）、Deutsch（德语）、Français（法语）、日本語（日语）、中文、Русский（俄语）、Українська（乌克兰语）、हिन्दी（印地语）。Arroxy 在首次启动时自动检测系统语言，随时可在工具栏的语言选择器中切换。翻译以纯 TypeScript 对象的形式存放在 src/shared/i18n/locales/ — 在 GitHub 上发个 PR 即可贡献。
需要装别的东西吗？
不需要。yt-dlp 和 ffmpeg 在首次启动时从它们的官方 GitHub releases 自动下载并缓存到本地。之后无需任何额外配置。
如果 YouTube 改了什么，还能用吗？
能 — Arroxy 有两层保障。第一，yt-dlp 是社区里最活跃维护的开源工具之一 — YouTube 一变，几小时内就更新。第二，Arroxy 完全不依赖 Cookie 或你的 Google 账号，所以没有会话过期，没有凭据要轮换。这两点结合让它比依赖浏览器导出 Cookie 的工具稳定得多。
能下播放列表吗？
目前支持单个视频。播放列表和频道支持在路线图里。
需要我的 YouTube 账号或 Cookie 吗？
不需要 — 这事比听起来更重要。大多数在 YouTube 更新后就罢工的工具会让你导出浏览器的 YouTube Cookie。这种方案每 30 分钟左右就坏一次（YouTube 会轮换会话），而 yt-dlp 自己的文档警告这可能让你的 Google 账号被标记。Arroxy 从不使用 Cookie 或凭据。无登录、无账号绑定，没东西过期，没东西被封。
macOS 提示 "应用已损坏" 或 "无法打开" — 怎么办？
这是 macOS Gatekeeper 在拦截未签名应用 — 并不是真的损坏。README 里有 macOS 首次启动的分步指引。
这合法吗？
为个人使用下载视频，在大多数地区一般是被接受的。你需要自己负责遵守 YouTube 的服务条款和当地法律。
安装

选择你的渠道。

直接下载或任意主流包管理器 — 每次发布都会自动更新。

Winget Windows

推荐用于 Windows 10/11。随系统自动更新。

winget install AntonioOrionus.Arroxy

Scoop Windows

通过 Scoop bucket 便携安装。无需管理员权限。

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

添加 cask，一行命令安装。通用二进制 (Intel + Apple Silicon)。

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

沙箱安装。从 Releases 下载 .flatpak 包，一行命令安装。无需配置 Flathub。

flatpak install --user Arroxy-*.flatpak
