Was es kann

Alles, was du erwartest — ganz ohne Reibung.

URL einfügen, Qualität wählen, Download klicken. Mehr nicht.

Bis zu 4K UHD

2160p, 1440p, 1080p, 720p — jede Auflösung, die YouTube bietet, plus Audio-only MP3, AAC und Opus.

60 fps & HDR erhalten

High-Framerate- und HDR-Streams kommen genau so durch, wie YouTube sie kodiert — ohne Qualitätsverlust.

Mehrere gleichzeitig

Reihe so viele Videos in die Warteschlange ein, wie du willst. Das Download-Panel verfolgt jeden Fortschritt parallel.

Auto-Updates

Arroxy hält yt-dlp und ffmpeg im Hintergrund aktuell — funktioniert nach jeder YouTube-Änderung.

Privatsphäre

Was Arroxy nicht tut.

Die meisten YouTube-Downloader fragen irgendwann nach deinen Cookies. Arroxy niemals.

Kein Login

Kein Google-Konto. Keine ablaufenden Sitzungen. Null Risiko, dass dein Konto markiert wird.

Keine Cookies

Arroxy fordert dieselben Tokens an wie jeder Browser. Nichts exportiert, nichts gespeichert.

Keine Telemetrie

Null Analytics. Deine Downloads, Verlauf und Dateien bleiben auf deinem Gerät — Punkt.

Keine Drittanbieter-Server

Die ganze Pipeline läuft lokal über yt-dlp + ffmpeg. Dateien berühren nie einen Remote-Server.

Installation

Wähle deinen Kanal.

Direkter Download oder ein gängiger Paketmanager — alle aktualisieren sich automatisch mit jedem Release.

Winget Windows

Empfohlen für Windows 10/11. Aktualisiert sich mit dem System.

winget install AntonioOrionus.Arroxy

Scoop Windows

Portable Installation via Scoop-Bucket. Keine Admin-Rechte nötig.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Cask hinzufügen, mit einem Befehl installieren. Universal Binary (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy
