Що він робить

Усе, чого ви очікуєте, без жодного тертя.

Вставте URL, оберіть якість, натисніть «Завантажити». Та й усе.

До 4K UHD

2160p, 1440p, 1080p, 720p — усі роздільні здатності, що пропонує YouTube, плюс лише аудіо у MP3, AAC та Opus.

60 fps і HDR збережено

Високочастотні та HDR-потоки приходять саме такими, як їх кодує YouTube — без втрати якості.

Декілька одночасно

Ставте в чергу скільки завгодно відео. Панель показує прогрес кожного паралельно.

Автооновлення

Arroxy сам підтримує yt-dlp та ffmpeg актуальними — працює після будь-яких змін YouTube.

9 мов

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी — визначає вашу автоматично.

Крос-платформний

Нативні збірки для Windows, macOS та Linux — інсталятор, портативна версія, DMG або AppImage.

Субтитри — як вам зручно

Ручні або авто-згенеровані субтитри у SRT, VTT чи ASS — поруч із відео, вбудовані в портативний .mkv або складені в підпапку Subtitles/.

SponsorBlock вбудовано

Пропускайте або позначайте сегменти спонсорів, вступи, кінцівки, самопіар тощо — вирізайте через FFmpeg або додавайте розділи. Ваш вибір, для кожної категорії.

Автозаповнення з буфера обміну

Скопіюйте посилання YouTube будь-де, і Arroxy виявить його при поверненні до застосунку — запит підтвердження зберігає контроль. Вмикайте та вимикайте в розширених налаштуваннях.

Автоочищення URL

Трекінгові параметри (si, pp, feature, utm_*, fbclid, gclid тощо) автоматично видаляються зі вставлених YouTube-посилань, а обгортки youtube.com/redirect розгортаються — у полі URL завжди відображається канонічна адреса.

Згортання в трей

Закриття вікна ховає Arroxy в системний трей. Завантаження тривають у фоні — натисніть значок трею, щоб повернути вікно, або виходьте через меню трею.

Вбудовані метадані й обкладинка

Назва, дата завантаження, виконавець, опис, обкладинка та маркери розділів записуються безпосередньо у файл — без додаткових файлів і ручного тегування.

У дії

Зроблено заради ясності, а не для безладу.

Приватність

Що Arroxy не робить.

Більшість завантажувачів YouTube рано чи пізно просять ваші cookies. Arroxy — ніколи.

Без входу

Без Google-акаунта. Без сесій, що закінчуються. Нульовий ризик блокування акаунта.

Без cookies

Arroxy просить ті самі токени, що й будь-який браузер. Нічого не експортується й не зберігається.

Без телеметрії

Жодної аналітики. Ваші завантаження, історія та файли залишаються на вашому пристрої — крапка.

Без сторонніх серверів

Уся обробка йде локально через yt-dlp + ffmpeg. Файли ніколи не торкаються віддаленого сервера.

FAQ

Часті запитання

Які якості відео можна завантажувати?
Будь-які, що пропонує YouTube — 4K UHD (2160p), 1440p QHD, 1080p Full HD, 720p, 480p, 360p та лише аудіо. Потоки з високою частотою кадрів (60 fps, 120 fps) і HDR зберігаються як є. Arroxy показує всі доступні формати і дає точно обрати потрібний.
Це справді безкоштовно?
Так. Ліцензія MIT. Без преміум-рівнів, без заблокованих функцій.
Якими мовами доступний Arroxy?
Дев'ятьма, з коробки: English, Español (іспанська), Deutsch (німецька), Français (французька), 日本語 (японська), 中文 (китайська), Русский (російська), Українська і हिन्दी (хінді). Arroxy автоматично визначає мову операційної системи при першому запуску, і ви можете перемикнутися будь-коли через вибір мови в панелі інструментів. Файли локалізації — прості TypeScript-об'єкти у src/shared/i18n/locales/ — відкрийте PR на GitHub, щоб зробити внесок.
Чи треба щось встановлювати?
Ні. yt-dlp і ffmpeg автоматично завантажуються при першому запуску з офіційних GitHub releases і кешуються на вашій машині. Після цього жодних додаткових налаштувань.
Чи працюватиме, якщо YouTube щось змінить?
Так — Arroxy має два рівні стійкості. По-перше, yt-dlp — один із найактивніше підтримуваних відкритих інструментів, оновлення виходять упродовж годин після змін YouTube. По-друге, Arroxy зовсім не залежить від кук чи вашого акаунта Google, тож немає сесій, що спливають, і немає облікових даних, які треба міняти. Це поєднання робить його значно стабільнішим за інструменти, що використовують експорт кук із браузера.
Чи можна завантажувати плейлисти?
Зараз підтримуються лише окремі відео. Підтримка плейлистів і каналів — у дорожній карті.
Чи потрібен мій YouTube-акаунт або куки?
Ні — і це важливіше, ніж може здатися. Більшість інструментів, які перестають працювати після оновлення YouTube, радять експортувати куки YouTube із браузера. Це обхід ламається кожні ~30 хвилин, коли YouTube змінює сесії, а в документації yt-dlp прямо попереджають, що це може призвести до прапорця на акаунті Google. Arroxy ніколи не використовує куки чи облікові дані. Ні логіну. Ні прив'язаного акаунта. Нічого не спливає, нічого не банять.
macOS каже «застосунок пошкоджено» або «не вдається відкрити» — що робити?
Це macOS Gatekeeper блокує непідписаний застосунок — реального пошкодження немає. У README є покрокова інструкція для першого запуску на macOS.
Чи це законно?
Завантаження відео для особистого використання в більшості юрисдикцій загалом допустиме. Ви самі відповідаєте за дотримання Умов використання YouTube і місцевого законодавства.
Встановлення

Оберіть канал.

Пряме завантаження або будь-який популярний менеджер пакетів — усі оновлюються автоматично з кожним релізом.

Winget Windows

Рекомендовано для Windows 10/11. Оновлюється разом із системою.

winget install AntonioOrionus.Arroxy

Scoop Windows

Портативна установка через Scoop bucket. Прав адміністратора не потрібно.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Додайте tap, встановіть однією командою. Universal Binary (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

Установка в пісочниці. Скачайте бандл .flatpak з Releases і встановіть однією командою. Налаштування Flathub не потрібне.

flatpak install --user Arroxy-*.flatpak

Пряме завантаження Усі

NSIS-інсталятор, портативний .exe, .dmg, .AppImage або .flatpak — прямо з GitHub Releases.
