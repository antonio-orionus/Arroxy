Qué hace

Todo lo que esperas, sin fricción.

Pega una URL, elige la calidad, haz clic en descargar. Así de simple.

Hasta 4K UHD

2160p, 1440p, 1080p, 720p — todas las resoluciones que ofrece YouTube, además de audio MP3, AAC y Opus.

60 fps y HDR preservados

Las transmisiones de alta tasa de fotogramas y HDR llegan tal como las codifica YouTube — sin pérdida de calidad.

Varios a la vez

Pon en cola tantos videos como quieras. El panel de descargas muestra el progreso de cada uno en paralelo.

Actualizaciones automáticas

Arroxy mantiene yt-dlp y ffmpeg al día — funciona con cada cambio de YouTube.

9 idiomas

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी — detecta el tuyo automáticamente.

Multiplataforma

Compilaciones nativas para Windows, macOS y Linux — instalador, portátil, DMG o AppImage.

Subtítulos a tu manera

Subtítulos manuales o automáticos en SRT, VTT o ASS — junto al video, integrados en un .mkv portátil, o en una subcarpeta Subtitles/.

Velo en acción

Diseñado para la claridad, no para el desorden.

Privacidad

Lo que Arroxy no hace.

La mayoría de descargadores de YouTube acaban pidiendo tus cookies. Arroxy nunca lo hará.

Sin inicio de sesión

Sin cuenta de Google. Sin sesiones que caducan. Cero riesgo de que marquen tu cuenta.

Sin cookies

Arroxy solicita los mismos tokens que cualquier navegador. Nada se exporta, nada se almacena.

Sin telemetría

Cero analítica. Tus descargas, historial y archivos se quedan en tu dispositivo — punto final.

Sin servidores de terceros

Todo el proceso se ejecuta localmente con yt-dlp + ffmpeg. Los archivos nunca tocan un servidor remoto.

Instalar

Elige tu canal.

Descarga directa o cualquier gestor de paquetes — todos se actualizan automáticamente con cada versión.

Winget Windows

Recomendado para Windows 10/11. Se actualiza con el sistema.

winget install AntonioOrionus.Arroxy

Scoop Windows

Instalación portátil vía Scoop bucket. No requiere permisos de administrador.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Añade el tap, instala con un comando. Binario universal (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy
