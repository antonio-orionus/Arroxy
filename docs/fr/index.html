Ce qu'il fait

Tout ce que vous attendez, sans la moindre friction.

Collez une URL, choisissez une qualité, cliquez sur télécharger. C'est tout.

Jusqu'à 4K UHD

2160p, 1440p, 1080p, 720p — toutes les résolutions proposées par YouTube, plus l'audio seul en MP3, AAC et Opus.

60 fps et HDR préservés

Les flux haute fréquence d'images et HDR passent exactement comme YouTube les encode — sans perte de qualité.

Plusieurs à la fois

Mettez autant de vidéos que vous voulez en file d'attente. Le panneau suit la progression de chacune en parallèle.

Mises à jour automatiques

Arroxy garde yt-dlp et ffmpeg à jour en coulisse — résiste à chaque changement de YouTube.

9 langues

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी — détecte la vôtre automatiquement.

Multiplateforme

Builds natifs pour Windows, macOS et Linux — installeur, portable, DMG ou AppImage.

Sous-titres comme tu veux

Sous-titres manuels ou auto-générés en SRT, VTT ou ASS — à côté de la vidéo, intégrés dans un .mkv portable, ou rangés dans un dossier Subtitles/.

Voyez-le en action

Conçu pour la clarté, pas l'encombrement.

Confidentialité

Ce qu'Arroxy ne fait pas.

La plupart des téléchargeurs YouTube finissent par demander vos cookies. Arroxy ne le fera jamais.

Sans connexion

Aucun compte Google. Aucune session à expirer. Zéro risque que votre compte soit signalé.

Sans cookies

Arroxy demande les mêmes tokens que n'importe quel navigateur. Rien n'est exporté, rien n'est stocké.

Sans télémétrie

Zéro analytique. Vos téléchargements, historique et fichiers restent sur votre appareil — point final.

Sans serveurs tiers

Tout le pipeline tourne en local via yt-dlp + ffmpeg. Les fichiers ne touchent jamais un serveur distant.

Installer

Choisissez votre canal.

Téléchargement direct ou tout gestionnaire de paquets majeur — tous mis à jour automatiquement à chaque release.

Winget Windows

Recommandé pour Windows 10/11. Se met à jour avec le système.

winget install AntonioOrionus.Arroxy

Scoop Windows

Installation portable via le bucket Scoop. Aucun droit admin requis.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Tap le cask, installation en une commande. Binaire universel (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy
