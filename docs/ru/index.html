До 4K UHD

2160p, 1440p, 1080p, 720p — все разрешения, которые предлагает YouTube, плюс только аудио в MP3, AAC и Opus.

60 fps и HDR без потерь

Высокая частота кадров и HDR-потоки приходят ровно так, как их кодирует YouTube — без потери качества.

Сразу несколько

Ставьте в очередь сколько угодно видео. Панель загрузок отслеживает прогресс каждого параллельно.

Автообновления

Arroxy сам поддерживает yt-dlp и ffmpeg в актуальном состоянии — работает после любых изменений YouTube.

Кросс-платформенный

Нативные сборки для Windows, macOS и Linux — установщик, портативная версия, DMG или AppImage.

Субтитры — как вам удобно

Ручные или авто-сгенерированные субтитры в SRT, VTT или ASS — рядом с видео, встроенные в портативный .mkv или сложенные в подпапку Subtitles/.

SponsorBlock встроен

Пропускайте или отмечайте сегменты спонсоров, вступления, концовки, самопиар и прочее — вырезайте через FFmpeg или добавляйте главы. Ваш выбор, по каждой категории.

Автозаполнение из буфера обмена

Скопируйте YouTube-ссылку где угодно, и Arroxy обнаружит её при возврате в приложение — запрос подтверждения сохраняет контроль. Включайте и выключайте в расширенных настройках.

Автоочистка URL

Трекинговые параметры (si, pp, feature, utm_*, fbclid, gclid и другие) автоматически удаляются из вставленных YouTube-ссылок, а обёртки youtube.com/redirect разворачиваются — в поле URL всегда отображается канонический адрес.

Приватность

Чего Arroxy не делает.

Большинство загрузчиков YouTube рано или поздно просят ваши cookies. Arroxy — никогда.

Без входа

Без Google-аккаунта. Без истекающих сессий. Нулевой риск, что аккаунт пометят.

Без cookies

Arroxy запрашивает те же токены, что любой браузер. Ничего не экспортируется, ничего не хранится.

Без телеметрии

Никакой аналитики. Ваши загрузки, история и файлы остаются на вашем устройстве — точка.

Без сторонних серверов

Весь конвейер работает локально через yt-dlp + ffmpeg. Файлы никогда не попадают на удалённый сервер.

FAQ

Часто задаваемые вопросы

Какие качества видео можно скачивать?
Всё, что предлагает YouTube — 4K UHD (2160p), 1440p QHD, 1080p Full HD, 720p, 480p, 360p и только аудио. Потоки с высокой частотой кадров (60 fps, 120 fps) и HDR сохраняются как есть. Arroxy показывает все доступные форматы и даёт точно выбрать нужный.
Это правда бесплатно?
Да. Лицензия MIT. Никаких премиум-уровней, никаких заблокированных функций.
На каких языках доступен Arroxy?
На девяти, прямо из коробки: English, Español (испанский), Deutsch (немецкий), Français (французский), 日本語 (японский), 中文 (китайский), Русский, Українська (украинский) и हिन्दी (хинди). Arroxy автоматически определяет язык системы при первом запуске, и вы можете переключиться в любой момент через выбор языка в панели инструментов. Файлы локализации — это простые TypeScript-объекты в src/shared/i18n/locales/ — откройте PR на GitHub, чтобы внести вклад.
Нужно что-то устанавливать?
Нет. yt-dlp и ffmpeg автоматически скачиваются при первом запуске с официальных GitHub releases и кешируются на вашей машине. После этого никакой настройки не требуется.
Будет ли работать, если YouTube что-то изменит?
Да — у Arroxy два уровня устойчивости. Во-первых, yt-dlp — один из самых активно поддерживаемых open-source инструментов, обновления выходят в течение часов после изменений YouTube. Во-вторых, Arroxy вообще не зависит от кук или вашего аккаунта Google, так что нет сессий, которые истекают, и нет учётных данных, которые надо менять. Эта комбинация делает его значительно стабильнее инструментов, использующих экспорт кук из браузера.
Можно ли скачивать плейлисты?
Сейчас поддерживаются только отдельные видео. Поддержка плейлистов и каналов — в дорожной карте.
Нужен ли мой YouTube-аккаунт или куки?
Нет — и это важнее, чем кажется. Большинство инструментов, которые перестают работать после обновления YouTube, советуют экспортировать куки YouTube из браузера. Этот обходной путь ломается каждые ~30 минут, когда YouTube меняет сессии, а в документации yt-dlp прямо предупреждают, что это может привести к флагу аккаунта Google. Arroxy никогда не использует ни куки, ни учётные данные. Никакого логина. Никакого привязанного аккаунта. Нечему истекать, нечему банить.
macOS говорит «приложение повреждено» или «не может быть открыто» — что делать?
Это macOS Gatekeeper блокирует неподписанное приложение — реального повреждения нет. В README есть пошаговая инструкция для первого запуска на macOS.
Это законно?
Скачивание видео для личного использования в большинстве юрисдикций обычно допустимо. Вы сами отвечаете за соблюдение Условий использования YouTube и местных законов.
Установка

Выберите канал.

Прямая загрузка или любой популярный менеджер пакетов — все обновляются автоматически с каждым релизом.

Winget Windows

Рекомендуется для Windows 10/11. Обновляется вместе с системой.

winget install AntonioOrionus.Arroxy

Scoop Windows

Портативная установка через Scoop bucket. Права администратора не нужны.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Добавьте tap, установите одной командой. Universal Binary (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

Установка в песочнице. Скачайте бандл .flatpak из Releases и установите одной командой. Настройка Flathub не нужна.

flatpak install --user Arroxy-*.flatpak

NSIS-установщик, портативный .exe, .dmg, .AppImage или .flatpak — прямо из GitHub Releases.
